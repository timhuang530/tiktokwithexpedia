Place the homepage feed video here as:

- `public/media/feed-video.mp4`
- `public/media/cathy-avatar.png`

Recommended:

- vertical aspect ratio close to `9:16`
- duration `10-20s`
- encoded as `mp4 (H.264)`
- optimized for web playback

Avatar notes:

- use a square image
- the UI crops it to a circle automatically
- black corners outside the circular photo will be clipped
